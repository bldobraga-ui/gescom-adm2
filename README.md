# GESCOM ADM - Gestão Completa Administrativa

Sistema profissional para gestão de colaboradores, documentos e vagas de emprego.

## 🚀 Como rodar o projeto no seu PC

Siga os passos abaixo para instalar e executar o sistema:

### 1. Pré-requisitos
Certifique-se de ter o **Node.js** instalado (versão 18 ou superior recomendada).

### 2. Instalação
Abra o terminal na pasta do projeto e execute:
```bash
npm install
```

### 3. Execução
Para iniciar o sistema em modo de desenvolvimento (Frontend + Backend):
```bash
npm run dev
```
O sistema estará disponível em: `http://localhost:3000`

### 4. Acesso ao Sistema
Utilize as credenciais padrão:
- **Email:** `admin@gescom.com`
- **Senha:** `admin123`

## 🛠️ Tecnologias Utilizadas
- **Frontend:** React.js, Tailwind CSS, Lucide Icons, Framer Motion.
- **Backend:** Node.js, Express, Multer (Uploads).
- **Banco de Dados:** Sistema de arquivos JSON (não requer configuração de banco de dados externo para rodar).
